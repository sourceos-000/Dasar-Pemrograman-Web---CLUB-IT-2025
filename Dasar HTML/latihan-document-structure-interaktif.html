Latihan Interaktif: Struktur Dokumen HTML

Ubah kode di kiri lalu klik Run untuk melihat hasil di kanan. Cocok untuk latihan yang akan di-embed ke Notion atau dijalankan lokal.
Tugas: 1) Ubah <h1> menjadi "Halo, saya [Namamu]". 2) Tambahkan satu <p> berisi motivasi singkat. 3) Klik Run untuk melihat hasil.
Instruksi singkat:
  1. Edit teks <h1> sesuai namamu.
  2. Tambahkan paragraf baru <p> (contoh: "Saya suka belajar web!")
  3. Klik Run untuk melihat perubahan.
